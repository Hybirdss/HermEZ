# HermEZ

**Hermes Agent 한방 설치 마법사** — Claude가 처음부터 끝까지 해줍니다.

```bash
git clone https://github.com/hybirdss/HermEZ && cd HermEZ && claude .
```

그게 전부입니다.

**2. OS 선택 (클릭 한 번)**

```
  어떤 환경에서 설치하시나요?

  ● Mac                  별도 준비 없이 바로 설치 가능합니다.
  ○ Linux / WSL2         터미널이 이미 있는 환경입니다.
  ○ Windows (WSL2 없음)  WSL2를 먼저 설치해야 합니다. (5분)
  ○ Android (Termux)     Termux 앱이 필요합니다.
```
